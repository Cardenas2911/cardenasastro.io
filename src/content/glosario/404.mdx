**404** es un código de estado HTTP que indica que una página web no se ha encontrado en el servidor. Ocurre comúnmente cuando una URL está mal escrita, ha sido eliminada o movida sin redirección adecuada.

Por ejemplo, si entras a `tusitio.com/blog/estrategia` y la página ya no existe, verás un error 404. Es importante personalizar esta página para mejorar la experiencia del usuario y guiarlo hacia contenido útil.
