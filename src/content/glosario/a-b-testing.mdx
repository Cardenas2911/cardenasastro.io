**A/B testing** es una técnica de experimentación que compara dos versiones de un mismo elemento para determinar cuál funciona mejor. Se utiliza en páginas web, emails, anuncios o interfaces para optimizar conversiones.

Por ejemplo, puedes mostrar dos botones distintos (uno azul y otro rojo) a diferentes usuarios y medir cuál consigue más clics. Esto permite tomar decisiones basadas en datos reales.
