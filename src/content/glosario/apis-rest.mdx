**APIs REST** (Representational State Transfer) son interfaces que permiten la comunicación entre sistemas a través de HTTP. Son fundamentales en el desarrollo web moderno para conectar bases de datos, servicios y aplicaciones.

Por ejemplo, cuando un sitio web muestra productos de una tienda en línea, probablemente está usando una API REST para obtener esa información del servidor.
